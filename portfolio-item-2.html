Tank Wars Game

A multiplayer tank destroyer game

Link to GitHub Repo

The Tank Game is a tank destroyer game that I built using JPanel. Through using rectangles it is able to identify the placement of each tank and bullets respectively. This identification allows for collisions to be detected and for life count to decrement as each tank experiences a collisions. There is also three different types of powerups that can be added to the tanks to increase the speed, the health, or the damage being given to the opposing tank.
