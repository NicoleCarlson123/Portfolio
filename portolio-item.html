Cube Dash Game

A 3D single player strategy game

PLAY THE GAME!

Cube Dash is a 3D single player strategy game I built in Unity. This game uses C# to detect collisions and to increment a score based on the distance the cube has traveled. It also employs some animation for the level change and game over text
